## ▶️ How to Compile & Run

1. Navigate to the folder where the file is located.  
   Example:
   ```bash
   cd sheets/sheet1
````

2. Compile the Java file:

   ```bash
   javac Q1_PrintHello.java
   ```

3. Run the program:

   ```bash
   java Q1_PrintHello
   ```
